## [0.1.0] - 2025-06-21

### Added

- Functionality to parse a `.wif` file, and additionally parse some sections
    - Treadling
    - Threading
    - Tie Up
    - Lift Plan
    - Color Palette
- Ability to turn sequence sections into a `Vec`
- Ability to write back to a `.wif`
- async file writes/reads with `async` feature
